ポジションサイジング101：ゲームに残り続ける数学

7分で読めるSignal Pilot チーム

素晴らしいセットアップを見つけました。高確率、明確な無効化ポイント、完璧なエントリー。自信があるのでサイズを大きくします。

負けました。口座が15%の打撃を受けます。今、ブレイクイーブンに戻るだけで17.6%の利益が必要です。

これがほとんどのトレーダーが破綻する理由です。悪いトレードではなく、悪いサイジングのせいです。

なぜポジションサイズがエントリーより重要か

受け入れがたい真実：70%の勝率があっても口座を飛ばす可能性があります。逆に、40%の勝率でも一貫して利益を上げられます。

違いはポジションサイズです。

勝ちトレードが平均$100、負けトレードが平均$300なら、70%の勝率は：(0.7 x $100) - (0.3 x $300) = $70 - $90 = 1トレードあたり-$20。

ほとんどのトレードで勝ちながらお金を失っています。

固定パーセンテージ法

実際に機能する最もシンプルなアプローチ：各トレードで口座の固定パーセンテージをリスクにさらす。

ほとんどのプロトレーダーは1-2%を使用。1%を使いましょう。

計算式：

ポジションサイズ =（口座リスク）/（1株あたりのリスク）

例：

  • 口座：$50,000
  • トレードあたりのリスク：1% = $500
  • エントリー価格：$100
  • ストップロス：$95
  • 1株あたりのリスク：$5
  • ポジションサイズ：$500 / $5 = 100株

なぜ1-2%？

これは恣意的ではありません。生存のための数学です。

1%リスクなら、10回連続で負けても資本の90%が残ります。

5%リスクなら、10回の損失で資本の60%になります。回復には67%の利益が必要。

10%リスクなら、10回の損失で資本の35%になります。回復には186%が必要。

アンチマーチンゲール原則

マーチンゲールは損失後に倍賭けします。カジノがギャンブラーからお金を抜き取る方法です。

賢いトレーディングは逆：アンチマーチンゲール。勝っている時に多くリスク、負けている時に少なくリスク。

固定パーセンテージサイジングでは、これが自動的に起こります。

サイズを調整する時

固定パーセンテージはほとんどの状況に対応しますが、以下の場合は縮小を検討：

  • 相関： 相関のある資産での複数ポジション
  • ボラティリティスパイク： VIXが高い時
  • 不確実性： 決算、FRB会議、選挙
  • ドローダウン： ドローダウン中にリスクを減らすトレーダーもいる

要点

ポジションサイジングは、悪いトレードが教訓になるか災害になるかの違いです。

ルールはシンプル：1トレードあたり最大1-2%のリスク、ストップからサイズを計算、数学にエクスポージャーを自動スケールさせる。

間違いから回復することはできる。破産から回復することはできない。それに応じてサイズを決めよう。

品質スコアリングシステムはサイジングの決定をさらに知らせることができる。強いコンフルエンスを持つセットアップ—サイクルフェーズの整列、ボリューム確認、複数のインジケーターの一致—はフルポジションサイズを正当化するかもしれない。相反するシグナルを持つマージナルなセットアップはハーフサイズまたはパスを正当化するかもしれない。データドリブンのサイジングは方程式から感情を取り除く。
